CONTENT CATEGORIES OF HTML

1.Metadata Content

▹ is content that sets up the presentation or behavior of the rest of the content, or that sets up the relationship of the document with other documents, or that conveys other "out of band" information.

2.Flow Content

▹ Most elements that are used in the body of documents and applications are categorized as flow content.

3.Sectioning Content

▹ Most elements that are used in the body of documents and applications are categorized as flow content.

4.Heading Content

▹ Defines the header of a section (whether explicitly marked up using sectioning content elements, or implied by the heading content itself)

5.Phrasing Content

▹ The text of the document, as well as elements that mark up that text at the intra-paragraph level. Runs of phrasing content form paragraphs.

A few other elements belong to this category, but only if a specific condition is fulfilled:

  • area tag, if it is a descendant of a map element
  • del tag, if it is a descendant of a map element
  • link tag, if the itemprop attribute is present
  • map tag, if it contains only phrasing content
  • meta tag, if the itemprop

6.Embedded Content

▹ It imports another resource or inserts content content from another mark-up language or namespace into the document.

HTML ATTRIBUTES

1.The lang Attribute

▹ The language of the document can be declared in the html tag.

▹ The language is declared with the lang attribute.

▹ This attribute specifies the language to be used for a certain element.

2.The title Attribute

▹ A title attribute is added to the p element

▹ The title attribute helps describe information of a certain element.

3.The href Attribute

▹ HTML links are defined with the a tag.

▹ This attribute specifies the link and from where it originated.

4.Size Attribute

▹ HTML images are defined with the img tag.

▹ The filename of the source (src), and the size of the image (width and height) are all provided as attributes

▹ This attribute specifies the number of attributes in a certain element such as characters.

5.The alt Attribute

▹ The alt attribute specifies an alternative text to be used, when an image cannot be displayed.

▹ The value of the attribute can be read by screen readers. This way, someone "listening" to the webpage, e.g. a blind person, can "hear" the element.

▹ This attribute specifies the alternative text and would be applied if the text element is not working or compatible.

Selector Syntax

BASIC SELECTORS

1.Type Selector

▹ CSS type selectors match elements by node name. Used alone, therefore, a type selector for a particular node name selects all elements of that type — that is, with that node name — in the document.

2.Universal Selector

▹ An asterisk (*) is the universal selector for CSS. It matches a single element of any type. Omitting the asterisk with simple selectors has the same effect.

3.Attribute Selector

▹ Attribute selectors select an element using the presence of a given attribute or attribute value.

  • [attr]

    • ▹ Represents an element with an attribute name of attr.

  • [attribute=value]

    • ▹ Selects all elements with target="_blank"

  • [attribute~=value]

    • ▹ Selects all elements with a title attribute containing the value you put inside the attribute

  • [attribute !=value]

    • ▹ Select elements that either don't have the specified attribute.

4.ID Selector

▹ It matches an element based on the contents of that element's id attribute, which must match exactly the value given in the selector.

5.Class Selector

▹ CSS class selectors match an element based on the contents of the element's class attribute.The class attribute is defined as a space-separated list of items, and one of those items must match exactly the class name given in the selector.
