THE INTERNET & THE WORLD WIDE WEB

WHAT IS INTERNET?

Internet is a system of interconnected computer networks, a global medium wherein computers are connected together via standardized communication protocols such as TCP/IP. Any user can get information, if permission is granted, from one device on the fly.

1.Logically linked together by a globally unique address space based on the Internet Protocol(IP)

2.Able to support communications using the transmission control protocol/Internet Protocol

3.Provides uses or makes accessible either publicly or privately.

▹ once you connect two devices, it opens a lot of opportunities

▷ INTER ▹ to cross boundaries

▷ NETWORK ▹ means connecting devices

INTERNET OF THINGS (IoT)

HYPERTEXT TRANSFER PROTOCOL

HTTP (HyperText Transfer Protocol)

▹ Application layer communications protocol used to access resources(hypertext/hypermedia) on the World Wide Web.

▹ It was invented by Tim Berners-Lee.

▹ Jointly developed by the W3C and the IETF (Internet Engineering Task Force).

▹ It runs on TCP/IP protocol using port 80 by default, the secure version being TCP 443 (HTTPS, integrated TLS/SSL)

What is HTTP based on?

▹ HTTP is based on client-server architecture a.k.a clients, user-agents (web browsers, web crawlers, other end user tools and applications)
